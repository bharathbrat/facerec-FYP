Design and Analysis of Face Recognition Algorithms

by - Aakanksha Parameshwar (1BY10CS002), Bharath M S (1BY10CS015)

Face Recognition is the identification of a face as a known face or an unlnown face. In this project, we implement the various face recognition algorithms and compare them based on their time complexity, accuracy in matching, the various challenges they overcome, etc.

We have used 2 standard databases, namely Yale Database A, ORL Database for our project. The algorithms that we have implemented are:

  • Principle Component Analysis (PCA)
  • Linear Discriminant Analysis (LDA)
  • Scale Invariant Feature Tranform (SIFT)
  • Line Edge Map (LEM)
