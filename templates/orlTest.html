Design and Analysis of Face Recognition Algorithms

The tests available on ORL database are:

  • General test - First 5 images (i.e '1','2','3','4','5') are put into training dataset and the next 5 images (i.e. '6','7','8','9','10') into testing dataset.
  • Pose test - Normal images with no pose variation are put into training dataset (i.e. '1','3','7','8') and images with slight pose variation are put into testing dataset (i.e. '2','4','5','6','9','10').
  • Single Image test - This test gives the user the choice to select an image from the testing dataset, that contains the odd numbered images (i.e. '1','3','5','7','9'), which will be compared with other images in training dataset(i.e. '2','4','6','8','10') to give a success or failure.
